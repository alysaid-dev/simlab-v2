- **Status pembayaran denda:** `UNPAID` / `PAID` / `WAIVED` — istilah resmi "**Dibebaskan**" (bukan dispensasi/diputihkan).
- **Toleransi hari:** grace period setelah `endDate`. `0` = telat langsung esok harinya. `1` = boleh kembali H+1 tanpa denda.

Disimpan juga ke memory `lab_domain_rules.md`.

- `Room`: tambah `laboratoryId` (nullable) + relasi.
- `RoomReservation`: tambah `laboratoryId` (nullable) + `rejectionReason` + relasi.

### 3.3 Seed (`backend/prisma/seed.ts`)

Tambah `seedLaboratories()` (upsert by `name`) dan `seedAppSettings()` (upsert id=singleton). Idempotent — tidak overwrite assignment kepala/laboran yang sudah diset via UI.

### 3.4 API baru

**`/api/app-settings`**
- `GET` — semua user ter-auth (readonly).
- `PATCH` — `SUPER_ADMIN` only. Body zod: `{ lateFeePerDayIdr?, lateFeeToleranceDays? }`.

**`/api/laboratories`**
- `GET /` — list 3 lab + kepala + laborans.
- `GET /:id` — detail.
- `PATCH /:id` — `SUPER_ADMIN` only. Body: `{ description?, isActive?, kepalaLabId?, laboranIds? }`. Laboran replace-all (hapus semua lalu insert kalau `laboranIds` dikirim).

Files: `services/appSettings.service.ts`, `controllers/appSettings.controller.ts`, `routes/appSettings.routes.ts` (+ equivalent untuk laboratories). Terdaftar di `routes/index.ts`.
